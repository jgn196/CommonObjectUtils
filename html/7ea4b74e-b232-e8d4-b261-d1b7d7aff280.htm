Remarks

This class allows you to add groups of objects that should be equal in a fluent style.

This is a partial port of the Guava EqualsTester class.

Examples
Copy
new EqualsTester()
    .AddEqualityGroup("hello", "h" + "ello")
    .AddEqualityGroup(3, 1 + 2)
    .TestEquals();
